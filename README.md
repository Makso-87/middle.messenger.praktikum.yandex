**Это учебный проект курса "Middle frontend разработчик" от Яндекс.практикум**

## Команды проекта
### Запуск в режиме разработчика
```
npm run dev
```

### Запуск локального сервера
```
npm run start
```

### Запуск продуктовой сборки проекта
```
npm run build
```
